Intoduction

Cette application convertit un fichier de profil LIDAR exporté du site sitn.ne.ch en un script AutoCAD.

Création fichier source

Ouvrir le site du SITN dans un navigateur internet. Zoomer sur la zone à mesurer. A l'aide de l'outils "Profil LIDAR" tracer une ligne de profil.

Si l'outil de dessin n'est pas actif, cliquer sur le bouton "Dessiner un profil LIDAR".

Faire un double clic pour terminer le tracé du profil. Ensuite exporter le fichier au format CSV.
